Setup

Install & use

Two minutes from download to a filter in your game.

1

Download the installer (recommended) or the portable version — or skip the download entirely and use the web app.

2

Windows SmartScreen may warn on first run — the build isn't code-signed yet. Click More info → Run anyway. The desktop app keeps itself up to date from GitHub releases.

3

Build your filter, then export it from the top bar into Documents\My Games\Path of Exile 2 and select it in-game under Options → Game → Loot Filter.

4

Want the overlay? Set PoE2 to Borderless or Windowed (it can't draw over exclusive Fullscreen), enable it in Settings → Game Overlay, and toggle with Shift+Alt+F.

5

Found a bug or want a feature? Tell us on Discord or open an issue on GitHub.

FAQ

Honest answers

Is it free?

Yes. The web app is free to use and the Windows app is a free download from GitHub releases. No accounts, no paywalls.

Do I need the desktop app?

No — the browser version builds, imports and exports filters just fine. The desktop app adds the things a browser can't do: the in-game overlay, live Price Check through your own trade session, the live Market, game-log tracking for Campaign & Speedrun mode, offline use and auto-updates.

Can I get banned for using this?

Loot filters are an official game feature — the app writes standard PoE2 filter syntax that you load through the game's own options menu. The overlay is a normal always-on-top window; nothing reads game memory or automates gameplay. Price Check talks to the official trade website with your own session, and Campaign tracking reads the game's log file.

Why does Windows warn when I run the installer?

The build isn't code-signed (certificates are expensive), so SmartScreen shows its generic warning for unrecognised apps. The code is public on GitHub if you'd like to review it, and every release is built from it.

Where do my filters live?

On your machine — filters and settings are stored locally in the app/browser. Sharing to the Community page is opt-in: only filters you explicitly publish are uploaded.

Where can I get help?

Join the Discord — feedback, bug reports and filter questions all welcome — or open a GitHub issue.
